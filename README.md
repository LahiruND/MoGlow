# MoGlow Website

A premium, multi-page website for MoGlow Candles, built with HTML, Tailwind CSS, and JavaScript.

## Features
- **Modern & Responsive Design**: Works on all devices (mobile, tablet, desktop).
- **Premium Aesthetics**: Dark theme with gold accents, glassmorphism effects, and smooth animations.
- **Product Showcase**: Grid layout for displaying products.
- **Contact Form**: Interactive contact section.
- **About Page**: Storytelling section with high-quality imagery.

## How to Run
Since this project uses static files and Tailwind CSS via CDN, you don't need to install anything.

1. Navigate to the project folder: `d:/MOGLOW/Website`
2. Open `index.html` in your web browser (Chrome, Edge, Firefox, etc.).

## Files
- `index.html`: Home page
- `products.html`: Shop page
- `about.html`: About Us page
- `contact.html`: Contact page

## Note
